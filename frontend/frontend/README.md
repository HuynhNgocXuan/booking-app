# Getting Started

## Prerequisites

- Nodejs
- MongoDB

## Backend

#### 1. **Environment file**:

Navigate to `backend` folder create `.env` file

```
PORT=

MONGO_URL=
```

#### 2. Install Backend Dependencies

```
cd backend
npm install
```

#### 3. Run the backend server
```
npm run dev
```
